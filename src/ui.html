Welcome:

LabelING will extract the copy from a sketch file or an imported file(json, csv). It will create a key value row following these rules :

  • The key are using the name of the elements used in the sketch file.
  • Labels are retrieve first from the top artboard in the artboard-list.
  • The Key are created using this pattern: ArtBoardName__Symbol_fieldName.
  • some keys are shared such as: 'tab', 'title', 'ContinueButton', 'NextButton', 'BackButton', 'DoneButton'.
  • Element name containing [x] will be skipped.
  • There may be some UTF-8 encoding errors, check your text for safety.
